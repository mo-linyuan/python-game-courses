# 学生同步课件说明

这份说明是给学生看的，目标只有一个：

每次我更新课件后，你都能把最新版同步到自己的电脑里。

## 第一次使用

### 方法一：使用 Git

如果你的电脑已经安装了 Git，在终端里输入：

```bash
git clone 你的仓库地址
```

下载完成后，进入项目文件夹：

```bash
cd Python
```

### 方法二：直接下载 ZIP

如果还没有安装 Git：

1. 打开老师的 GitHub 仓库页面
2. 点击 `Code`
3. 点击 `Download ZIP`
4. 解压到自己的电脑上

这种方法也能拿到课件，但以后更新会没有 `git pull` 方便。

## 以后怎么同步更新

如果你已经是通过 Git 下载的课件，那么以后每次老师更新后，只需要：

1. 打开终端
2. 进入课件文件夹
3. 输入下面这条命令

```bash
git pull
```

这条命令的意思是：

把老师上传到 GitHub 的最新内容，拉到你自己的电脑里。

## 上课前建议做什么

每次上课前，先执行一次：

```bash
git pull
```

这样你打开的就是最新版课件。

## 写作业时的建议

不要直接把老师原来的课件改得乱七八糟，最好另存一个自己的版本。

例如：

- 老师文件：`lesson08_lists_backpack.ipynb`
- 你的文件：`lesson08_lists_backpack_student.ipynb`

## 如果出现问题

如果 `git pull` 后出现冲突，通常说明你改动了老师原来的文件。

最简单的做法是：

- 保留自己的练习文件
- 老师原版尽量不要直接覆盖修改

## 你真正要记住的只有一句话

第一次下载用：

```bash
git clone 仓库地址
```

以后更新用：

```bash
git pull
```
